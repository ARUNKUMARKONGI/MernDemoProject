To install react first node js should be installed in windows
Mern-> node js react using netlify and backend in render, database mongodb atlas

inside main project folder create 2 subfolders

mkdir frontend
cd frontend
npm create vite@latest frontend //this command will create frontend app using react
npm install
npm run dev  //to run the app everytime
npm install bootstrap



mkdir backend
cd backend
npm install
npm init -y 
npm install express mongoose cors   //to connect to the mongodb database
